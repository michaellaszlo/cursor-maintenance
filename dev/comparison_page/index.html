Comparison of cursor-maintenance implementations

This page shows the effects of several approaches to cursor maintenance on two formatting operations, commatize and trimify. You can alter the input text and cursor position in the input field below the name of each formatting operation in the following table.

The result of the formatting operation is displayed below the input field. The remaining cells show the cursor positions computed by various implementations of cursor maintenance.

The ad hoc, mock cursor, and meta implementations only return a new cursor position. The retrospective approach returns a numerical score for every possible cursor position in the formatted text. To view the scores computed by one of the retrospective formulas, split Levenshtein and balanced frequencies, click on the "scores" button next to the respective result cell.
