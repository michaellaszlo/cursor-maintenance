Comparison of cursor-maintenance implementations

This page compares the performance of several approaches to cursor maintenance on two formats, commatize and trimify.

For each format, there is an input field in the row labeled "before". The formatted text is displayed in the row labeled "after". The next six rows show the cursor positions computed by various implementations of cursor maintenance.

The first four implementations only return a cursor position. The last two also return numerical scores for every position in the formatted text. To view the scores, click on the button labeled "scores" next to the result field.
