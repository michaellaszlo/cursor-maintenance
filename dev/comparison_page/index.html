Comparison of cursor-maintenance implementations

This page shows the effects of several attempts at cursor maintenance on two formatting operations, commatize and specify. You can specify the input text for each formatting operation by editing the input field at the top of the respective column in the following table. You can also move the cursor to the position of your choosing.

The result of the formatting operation is displayed immediately below the input field. The remaining cells show the cursor positions computed by the various implementations of cursor maintenance. The ad hoc, mock cursor, and meta implementations only return a new cursor position. The retrospective approach is implemented using two formulas, split Levenshtein and balanced frequencies, which return a numerical score for every possible cursor position in the formatted text. To view the scores computed by a retrospective formula, click on the button labeled "scores" in the respective result cell.
