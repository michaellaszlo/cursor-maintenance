This page demonstrates two approaches to cursor maintenance. The problem is discussed in a detailed article.

Additional code and technical documents are available in a source repository.

Meta

The meta approach entails reimplementing a formatting operation by applying highly localized text operations to a text-with-cursor object. This object moves the cursor in a predictable manner after each local text operation.

The commatize formatting operation is only applicable to a string of digits and commas. The input field strips out all other characters before applying commatize. The formatted text displays a non-negative integer with commas separating each three-digit group from right to left.

The trimify formatting operation is applicable to any text. It removes whitespace from the beginning and end of the text, and condenses all other whitespace sequences to one character each.
