Notes

This page demonstrates two solutions to the problem of cursor maintenance, which is explored in a detailed article.

For additional code and documentation, please see the source repository.

The meta approach entails rewriting a formatting operation so that instead of operating on text, it operates on a text-with-cursor object. The object updates the cursor in a predictable manner for each elementary text operation.

The commatize formatting operation is only applicable to a string of digits and commas. The input field strips out all other input before applying commatize. The formatted text displays a non-negative integer with commas separating each three-digit group from right to left.

The trimify formatting operation is applicable to any text. It removes whitespace from the beginning and end of the text, and condenses all other whitespace sequences to one character each.
